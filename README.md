# VeriCart - Your Ultimate E-commerce Platform

VeriCart is a powerful and user-friendly e-commerce platform designed to provide a seamless shopping experience. Built with React, Node.js, and MongoDB, VeriCart enables businesses to launch and manage their online stores effortlessly.

### Prerequisites
Ensure you have the following installed:
- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
Clone the repository and install dependencies:
```sh
git clone https://github.com/isaacog12/vericart.git
cd vericart
npm install  # or yarn install
```

### Running the Application
To start the development server, run:
```sh
npm start  # or yarn start
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
